Profil Singkat Kota Cimahi

Cimahi mulai dikenal pada tahun 1811, Gubernur Jendral Willem Daendels membuat jalan Anyer - Panarukan, dengan dibuatnya pos penjagaan (IoJi) di Alun-alun Cimahi sekarang. Tahun 1874 - 1893, dilaksanakan pembuatan jalan kereta api Bandung - Cianjur sekaligus pembuatan stasiun kereta api Cimahi. Tahun 1886 dimulainya pembangunan pusat pendidikan militer dan fasilitas lainnya (RS Dustira, rumah tahanan militer, dll).
